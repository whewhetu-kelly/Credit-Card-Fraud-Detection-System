# Credit Card Fraud Detection System

A production-grade, rule-based Credit Card Fraud Detection System developed as an HND Computer Science project. The platform helps financial organizations monitor transactions, detect suspicious activities using configurable fraud rules, generate alerts, and manage fraud investigations through a secure role-based dashboard.

> **Note:** This repository showcases the project's features, architecture, and documentation. The complete source code is private.

---

## Features

- Secure User Authentication
- Role-Based Access Control (Admin, Analyst, Viewer)
- Transaction Management
- Rule-Based Fraud Detection
- Real-Time Fraud Alerts
- Transaction Search & Filtering
- Fraud Investigation Workflow
- Administrative Dashboard
- Analytics & Reports
- Audit Logging
- CSV Dataset Import
- Responsive User Interface

5. Configure the web server.
6. Set folder permissions.
